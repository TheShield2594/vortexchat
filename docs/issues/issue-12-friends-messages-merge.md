# Consider merging Friends into Messages tab on mobile

**Labels:** enhancement

## Context
Currently Friends is a separate sub-tab within the DM list on mobile. The original navigation UX analysis proposed merging Friends into the Messages tab to reduce navigation depth.

## Source
Previously tracked in `docs/navigation-ux-analysis.md`, now deleted. The current approach (separate Friends sub-tab in DM list) works but may add unnecessary navigation depth on mobile.

## Notes
This is a design decision — the current implementation is functional. Evaluate whether merging would improve the mobile UX or if the current approach is sufficient.
